# Blogger Adsterra Template Optimization

Repository ini digunakan untuk optimasi template Blogger dengan integrasi iklan Adsterra.

## Target Optimasi

- Homepage ads setelah posting ke-4 dan ke-8
- In-article ads:
  - bawah judul
  - paragraf ke-4
  - sebelum related post
  - atas komentar
- Mengurangi script iklan berulang
- Mempertahankan tampilan dan fitur template
